# 11. Открытые решения

Codex не принимает эти продуктовые решения самостоятельно:

- платёжный провайдер;
- проверка ИП/ТОО;
- категории и запрещённые товары;
- точные file/AI limits;
- сроки хранения;
- OCR/LLM provider;
- рейтинг;
- уведомления;
- русский/казахский интерфейс;
- юридические документы;
- налоговый учёт;
- production hosting.

Использовать конфигурируемые порты и минимальные безопасные defaults без преждевременного усложнения.
