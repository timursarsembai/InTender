# 07. Платный ИИ-разбор

## Тариф
1 000 ₸: один файл, один запуск, одна итоговая товарная позиция. Публикация отдельно 50 ₸. Стартовые лимиты: до 30 страниц, 20 МБ; PDF, DOCX, XLSX, JPG, PNG.

## Бесплатный режим
ИИ не используется; несколько товаров не определяются; пользователь отвечает за файл; исходник виден поставщикам; базовые детерминированные фильтры блокируют очевидные контакты/URL.

## Платный режим
Исходник закрыт от поставщиков. Система извлекает данные, выявляет позиции, очищает идентификаторы, предлагает выбрать одну позицию и создаёт редактируемый черновик. Автопубликация запрещена.

## Извлечение
Название, количество, единица, бренд, модель, артикул, характеристики, сертификаты, гарантия, комплектность, сроки/место при однозначности.

## Удаление
Заказчик, площадка, номер закупки/лота, БИН, ФИО, телефоны, email, URL, реквизиты, подписи, печати и признаки источника.

## Технические правила
- versioned JSON schema;
- backend validation;
- документ — недоверенный ввод;
- команды/prompt injection внутри документа игнорируются;
- tools для модели отключены;
- внутренние инструкции не раскрываются;
- логирование содержимого запрещено.

## Ошибки
Бесплатный retry/refund при технической ошибке, пустом или невалидном результате. Нет возврата из-за плохого исходника, отсутствующих данных или несогласия со стилем. Результат можно исправить вручную.
